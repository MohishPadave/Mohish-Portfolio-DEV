# Project Images Organization

This folder contains images for each project collection in the Photos app.

## Folder Structure

```
projects/
├── project1/    # E-Commerce Platform
├── project2/    # Social Media Dashboard
├── project3/    # Weather App
├── project4/    # Task Manager
├── project5/    # Portfolio Website
├── project6/    # Chat Application
├── project7/    # Blog Platform
├── project8/    # Music Player
├── project9/    # Fitness Tracker
└── project10/   # Recipe App
```

## How to Add Images

1. **Place your project screenshots** in the corresponding project folder
   - Example: `project1/screenshot1.png`, `project1/screenshot2.jpg`, etc.

2. **Supported formats**: JPG, PNG, JPEG, WebP

3. **Naming convention**: Use descriptive names
   - `homepage.png`
   - `dashboard.jpg`
   - `mobile-view.png`
   - `feature-detail.jpg`

4. **Image recommendations**:
   - Resolution: 1920x1080 or higher for best quality
   - File size: Keep under 2MB per image for faster loading
   - Use screenshots of your actual projects

## Example

For your E-Commerce Platform (project1):
```
project1/
├── homepage.png
├── product-page.png
├── cart.png
└── checkout.png
```

The Photos app will automatically display all images from each project folder when you click on that collection.
