
#  macOS Portfolio

A clean, modern, and fully interactive macOS-inspired portfolio built using **React**, **Tailwind CSS**, and **GSAP**.
Designed to feel like a real Mac desktop — smooth, playful, and professional.

##  Overview

This project recreates the familiar macOS Big Sur interface and turns it into a personal portfolio experience. From draggable windows to a functional desktop, everything is built to feel dynamic and alive.

##  Features

* **macOS Big Sur UI** — glassmorphism, dock, folders, and vivid blur effects
* **Login Screen** — custom lock screen with guest mode
* **Interactive Desktop** — draggable/resizable windows, openable apps
* **Functional Icons** — folders, browser window, external links, and more
* **GSAP-powered Animations** — smooth transitions, easing, and micro-interactions
* **Fully Responsive** — works on all screen sizes
* **Easy to Customize** — plug in your own projects, resume, and details

##  Customization

This project is built to be *your* portfolio, so everything is easy to edit:

###  Windows Content

Modify `src/components/Window.jsx` to update:

* About Me
* Project showcase
* Tech stack
* Resume link
* Contact information

### Desktop Icons

Add, remove, or change desktop icons by editing:
`src/components/Desktop.jsx`

##  Tech Stack

* **React** — component-driven UI
* **Tailwind CSS** — rapid styling with utility classes
* **GSAP** — world-class animation library
* **Vite** — blazing-fast build tooling
